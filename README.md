# 🧮 Calculadora

Aplicação desenvolvida em React e TypeScript inspirada na Calculadora do Windows, com foco na prática de componentização, gerenciamento de estado, manipulação de eventos e lógica de operações matemáticas.

A aplicação realiza as operações básicas de uma calculadora de desktop, incluindo suporte ao teclado físico e tratamento de casos especiais, como divisão por zero.

## 🚀 Funcionalidades

- ✅ Operações de adição, subtração, multiplicação e divisão
- ✅ Entrada de números decimais
- ✅ Alteração de sinal (±)
- ✅ Limpeza parcial (CE) e limpeza completa (C)
- ✅ Backspace
- ✅ Encadeamento de operações
- ✅ Repetição da última operação ao pressionar "="
- ✅ Tratamento de divisão por zero
- ✅ Suporte ao teclado físico

## 🧠 Conceitos aplicados

Projeto desenvolvido com foco na prática de fundamentos do React e TypeScript.

- Componentização
- Props
- Hooks (`useState` e `useEffect`)
- Gerenciamento de estado
- Manipulação de eventos
- Renderização dinâmica

- Organização de componentes
- Lógica de negócio separada da interface

## ▶️ Como executar

### Pré-requisitos

- Node.js instalado

### Clonar o repositório

```bash
git clone https://github.com/rafaelvassis/calculadora-React.git
cd calculadora-React
```

### Instalar dependências

```bash
npm install
```

### Executar

```bash
npm run dev
```
